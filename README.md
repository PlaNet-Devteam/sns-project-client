## Git

### 🌿 Branch

- main : 서비스 운영 브랜치
- dev : 개발중인 브랜치
- feat/기능명 : 기능 개발 브랜치

<br/>

### 💌 Commit Message

| Message  |                             설명                             |
| :------: | :----------------------------------------------------------: |
|   init   |                          초기 설정                           |
|   feat   |                       새로운 기능 추가                       |
|   fix    |                          버그 수정                           |
|   docs   |                          문서 수정                           |
| refactor |                        코드 리팩터링                         |
|   test   | 테스트 코드, 리팩터링 테스트 코드 추가(프로덕션 코드 변경 X) |
|  chore   |   빌드 업무 수정, 패키지 매니저 수정(프로덕션 코드 변경 X)   |
|  rename  |      파일 혹은 폴더명을 수정하거나 옮기는 작업만인 경우      |
